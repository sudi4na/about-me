# Selamat Datang di Sudiana Github Site.

Selamat datang saya ucapkan karena anda sudah meluangkan waktu untuk berkunjung ke website saya pribadi. Saya merupakan seorang warga indonesia yang tinggal di sebelah timur pulau bali. Tepatnya sayatinggal di karangasem Bali.Keseharian saya lewatkan di luar ruangan dan malamnya saya habiskan di depan komputer. Saya bukanlah seorang yang sukses tetapi berusaha untuk sukses. Berbekal pengetahuan dan pengalaman yang saya dapatkan selama bekerja. Sekarang saya sedang menggeluti bisnis wirausaha saya di bidang IT dan Teknologi. Ada banyak pekerjaan yang sudah saya geluti antara lain Website designer, IT developer, IT consultan, dan juga mengelola beberapa website milik saya pribadi.

Website Designer

Meski saya bukan jebolan Informatika atupun Ilmu komputer, tetapi saya menyukai pekerjaan ini. Bebekal pengetahuan mengenai HTML, CSS, PHP, dan Wordpress, sudah lumayan banyak website yang sudah saya ciptakan antara lain Website Baguz Info Media ( http://baguz.net ), Balinetcom ( http://balinetcom.com ), Amedcafe (http://amedcafe.com ), Amedtours (amedtours.com ), dan masih banyak lagi yang lainya. Semua website itu saya buat menggunakan CMS terpopuler yaitu wordpress. Selain familiar dengan CMS ini, saya juga dengan mudah dapat membuat Design website menggunakan Wordpress.

IT developer dan IT consultan

Sebelumnya saya memiliki basic pengalamandi bidang network IT dan Komputer. karena itulah saya memutuskan untuk melanjutkan pengalaman itu menjadi bisnis pribadi saya. Meskipun pendapatan tidak begitu banyak, tetapi tetap saja saya jalankan. karena Disitulah Hoby dan pengalaman yang saya miliki.Untuk sekarang pelanggan saya cuma sekitar Wilayah karangasem saja.Untuk selanjutnya saya berkeinginan untuk memperluas wilayah usaha, dengan melihat aspek submer daya manusia yang bisa saya ajak bekerja sama.

Authors and Contributors

Saya memang tidak memiliki banyak waktu untuk melakukan banyak pekerjaan, tetapi saya berusaha membagi waktu untuk bisa berbagi pengalaman melalui website bersama baguz Info media ( Baguz.net ). Semuanya saya lakukan untuk bisa sharing pengalaman maupun sharing pengetahuan yang saya dapatkan selama saya bekerja.
